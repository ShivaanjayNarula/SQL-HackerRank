# SQL HackerRank Solutions

This repository contains my personal solutions to various SQL challenges on **HackerRank**. The goal is to document my progress, refine my database querying skills, and provide a reference for different SQL dialects.

---

## Repository Structure

The challenges are organized by their difficulty level and category. Each folder (or file) corresponds to a specific problem set on the platform.

| Category | Difficulty |
| :--- | :--- |
| **Basic Select** | Easy |
| **Advanced Select** | Medium |
| **Aggregation** | Easy/Medium |
| **Basic Join** | Medium |
| **Advanced Join** | Hard |
| **Alternative Queries** | Medium |

---

## 🛠️ Languages & Tools

* **Dialects used:** MySQL / MS SQL Server / Oracle
* **Platform:** [HackerRank SQL Dashboard](https://www.hackerrank.com/domains/sql)
* **Concepts Covered:** * Joins (Inner, Left, Right, Full)
    * Subqueries & Common Table Expressions (CTEs)
    * Window Functions (`RANK`, `DENSE_RANK`, `ROW_NUMBER`)
    * Aggregate Functions (`SUM`, `AVG`, `COUNT`, `MIN/MAX`)
    * String Manipulation & Regular Expressions

---

## How to Use This Repo

1.  **Browse by Difficulty:** Navigate through the directories to find specific problems.
2.  **View Solutions:** Each file contains the SQL query required to solve the problem.
3.  **Run the Code:** You can copy these queries and run them directly in the HackerRank editor to see the results.
